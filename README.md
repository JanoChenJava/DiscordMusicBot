# Music-Bot
> Simple music bot 

## Install

```bash
npm i
```

## Usage

Edit the config.js file:

```js
exports.TOKEN = ''; // Your token here

exports.PREFIX = 'm!'; // Your preferred prefix here

exports.GOOGLE_API_KEY = ''; // Your youtube/google API key here
```
